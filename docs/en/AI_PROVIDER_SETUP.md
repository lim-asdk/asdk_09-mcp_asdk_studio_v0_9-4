# AI Provider Setup Guide (English)

ASDK Studio supports various AI providers through OpenAI-compatible endpoints.

## 1. Getting an API Key
You need an API key from one of the following providers:
- **xAI (Grok)**: [https://console.x.ai/](https://console.x.ai/)
- **Google (Gemini)**: [https://aistudio.google.com/](https://aistudio.google.com/)
- **OpenAI (GPT)**: [https://platform.openai.com/](https://platform.openai.com/)
- **DeepSeek**: [https://platform.deepseek.com/](https://platform.deepseek.com/)

## 2. Setting up in Studio
1. Open **Settings** > **AI Profiles**.
2. Select your profile (default is `default`).
3. Enter your **API Key** and **Model ID**.
4. Set the **Base URL** (e.g., `https://api.x.ai/v1`).
5. Click **Test Key** to verify.
6. Click **Save** to apply.

## 3. Local Storage Policy
Your keys are saved locally in the `user_data/` folder and are never sent to our servers or included in Git.
