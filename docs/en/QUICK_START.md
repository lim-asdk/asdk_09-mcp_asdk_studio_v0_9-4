# Quick Start

If you want the full walkthrough with screenshots, open the full guide first:
[Setup Guide](SETUP_GUIDE.md)

## 1. Clone

```bash
git clone https://github.com/lim-asdk/asdk_09-mcp_asdk_studio_v0_9-4.git
cd asdk_09-mcp_asdk_studio_v0_9-4
```

## 2. Launch

```bash
python main.py
```

## 3. Basic Rule

- Store private profiles and keys in `user_data/`.
- Do not commit `user_data/` or `.env`.
